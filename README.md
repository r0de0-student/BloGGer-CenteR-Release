# BloGGing CenteR

**Версия: 0.13.4**

Платформа для ведения тематических блогов. Система позволяет авторам создавать блоги и публиковать статьи, а так же комментировать статьи. Администратор управляет пользователями через админ-панель.



## 🚀 Функционал

### Для всех пользователей
- Регистрация и авторизация
- Просмотр статей и комментариев

### Для автора
- Создание и управление блогом
- Публикация, редактирование и удаление статей
- Просмотр личной статистики (блоги, статьи, просмотры, комментарии)

### Для администратора
- Управление пользователями (блокировка/разблокировка)
- Просмотр системных отчётов:
  - Пользователи системы
  - Статьи и просмотры
  - Топ блогов
  - Выгрузка в **Excel (.xlsx)** через PhpSpreadsheet
  - Выгрузка в **Word (.docx)** через PhpWord



## 🛠 Технологии

Технология 
**Backend** PHP 8.0+ 
**Database**  MySQL 5.7+ 
**Web Server** Apache 2.4+ XAMPP
**Frontend** HTML5, CSS3 
**Security** PDO (prepared statements), bcrypt (хэширование паролей), XSS-защита 



## 📦 Установка и запуск

### 1. Требования
- Установленный **XAMPP** (рекомендуется) 
- PHP 8.0 или выше
- MySQL 5.7 или выше
- Composer (для установки зависимостей)

### 2. Клонирование репозитория
```bash
git clone https://github.com/r0de0-student/BloGGer-CenteR.git
ИЛИ:
 - Открыть папку Project
 - Найти там model-3
 - И перенести в заранее подготовленную папку /xampp/htdocs/blogger-center/
 - Запустить Apache и MySQL 
 - Проверить работу веб-приложения по ссылке http://localhost/blogger-center/Project/module-3/?action=home
```
