## 📖 О проекте

**BloGGing CenteR** — это веб-платформа, созданная для тех, кто хочет делиться своими мыслями, идеями и опытом через блоги. Проект ориентирован на качественный текстовый контент, с поиском нужной вам информацией или для простого интереса от изучения чего то нового до обычного общения с другими пользователями.

### Цели проекта:
- Создать удобную платформу для ведения блогов
- Обеспечить читателям доступ к интересному контенту
- Построить сообщество вокруг обсуждения статей

## ✨ Функциональность

### Для всех пользователей:
- Просмотр статей на главной странице
- Поиск статей по ключевым словам
- Регистрация и аутентификация

### Для авторов:
- Создание и управление своим блогом
- Публикация, редактирование и удаление статей
- Удаление комментариев под своими статьями
- Статистика блога

### Для администраторов:
- Управление пользователями (блокировка/разблокировка)
- Изменение ролей пользователей
- Удаление любых статей и комментариев
- Модерация контента

### Дополнительно:
- Защита от SQL-инъекций (PDO prepare)
- Защита от XSS (htmlspecialchars)
- Единая точка входа (Front Controller)
